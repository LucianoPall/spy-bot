================================================================================
ESTRUTURA DO BANCO DE DADOS
================================================================================

TABELA: spybot_campaigns
  - id (UUID, PK)
  - user_id (UUID, FK) - Isolamento por usuário
  - name (VARCHAR, UNIQUE)
  - description, icon_emoji, color_hex
  - status (DEFAULT 'active')
  - created_at, updated_at (TIMESTAMP)

TABELA: spybot_campaign_clones
  - id (UUID, PK)
  - campaign_id (UUID, FK)
  - clone_id (UUID, FK)
  - is_favorite (BOOLEAN)
  - performance_notes, tags (TEXT[])
  - position (INT)
  - added_at, updated_at (TIMESTAMP)

TABELA: spybot_campaign_stats
  - id (UUID, PK)
  - campaign_id (UUID, FK UNIQUE)
  - user_id (UUID, FK)
  - total_clones (INT)
  - favorite_clones (INT)
  - clones_with_high_ctr (INT)
  - avg_ctr (DECIMAL)
  - last_updated (TIMESTAMP)

TABELA MODIFICADA: spybot_generations
  - + campaign_id (UUID, FK)
  - + clone_tags (TEXT[])

SEGURANÇA IMPLEMENTADA

   - Usuário A SÓ EDITA campanhas dele

   - Constraints de unicidade
   - Validação de dados

   - Dados sempre consistentes
