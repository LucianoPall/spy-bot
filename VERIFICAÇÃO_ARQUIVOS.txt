TROUBLESHOOTING RÁPIDO:
================================================================================

Erro: "Cannot find module 'vitest'"
→ npm install

Erro: "Testes pendurados"
→ Verifique vi.clearAllTimers() em afterEach()

Erro: "@/* não resolve"
→ Confirme vitest.config.ts tem paths configurado

Erro: "Testes muito lentos"
→ Use npm run test:watch (modo incremental)

Mais ajuda? → Veja TESTING_GUIDE.md seção "Troubleshooting"
