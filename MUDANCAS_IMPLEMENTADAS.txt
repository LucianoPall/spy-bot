🎯 COMO FUNCIONA

Fluxo de Carregamento:

1. ✓ Verificar localStorage (100ms - super rápido)
2. ✗ Não encontrou?
   └─→ Chamar /api/get-image (com retry 3x)
3. ✗ Falhou?
   └─→ Tenta URL diretamente
4. ✗ Falhou tudo?
   └─→ Mostra placeholder cinza

Exemplo antes:
<img src={`/api/proxy-image?url=${clone.image1}`} />

Exemplo depois:
<img src={getImageUrl('image1')}
     onError={(e) => { e.currentTarget.src = PLACEHOLDER_IMAGE; }} />
